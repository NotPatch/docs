### **Player Statistics**

#### Basic Statistics

| Placeholder  | Description                  |
| ------------ | ---------------------------- |
| %has_coins%  | Total coins earned by player |
| %has_games%  | Total games player           |
| %has_wins%   | Number of games won          |
| %has_losses% | Number of games lost         |

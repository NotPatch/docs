\*\*NHideAndSeek \*\*is a modular, high-performance Hide and Seek plugin for Minecraft servers. It features fully configurable arenas, menus, statistics, and rewardsâ€”all managed through user-friendly YAML files. The plugin supports advanced features like PlaceholderAPI, economy integration, in-game chat, and detailed logging, making it ideal for both casual and competitive gameplay. NHideAndSeek is lightweight, production-ready, and perfect for any server looking to offer a customizable and engaging Hide and Seek experience.

# Installation

1. Download the NHideAndSeek plugin.
2. Place the `.jar` file into your server's `plugins` folder.
3. Restart the server (do not reload).
4. Get your license key from [**Our discord server**](discord.minebazaar.net)
4. Edit `license-key` in config.
5. Configure the plugin files located in `plugins/NHideAndSeek/`
